 即时标记是什么呢，比方说你从一个不想写HTML代码的人那里得到一个文本文件，希望把它转成HTML的格式来当作一个网页来使用。人工添加标签是繁琐的，因此想写一个程序来自动完成。这类问题称之为即时标记。
   考虑该工作的过程大概可以表述为：

将文本文件切分成段落，分块（被一个或多个空行隔开的段落称之为块block）
你应该能够处理不同的块。例如：标题，段落，列表等
对于不同的块抽象出不同的应用规则，添加相应的标记

因此程序应抽象出以下模块：

语法分析器。读取文本管理其他类的对象
规则。对应于不同的HTML标签，能够检测适用的块并进行格式化
过滤器。用于包装正则表达式，处理URL，邮箱地址等。
处理程序。语法分析器调用处理程序应用各种规则对文本进行格式化。为了方便扩展不同的处理程序能产生不同种类的标记。
